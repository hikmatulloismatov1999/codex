# Almost Equicontinuity and Translation Conditions

This folder contains a LaTeX paper draft about almost equicontinuity,
truncated translation conditions, and compactness in asymptotic \(L^p\)
spaces.

## Files

- `main.tex` - paper source
- `README.md` - project notes
- `.gitignore` - ignores LaTeX build output
- `Makefile` - optional build commands

## Build

If you have a LaTeX distribution installed, run:

```sh
make
```

or directly:

```sh
pdflatex main.tex
pdflatex main.tex
```
