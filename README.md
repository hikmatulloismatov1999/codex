# Almost Equicontinuity and Truncated Translation Conditions

This folder contains a polished LaTeX note on almost equicontinuity,
truncated translation conditions, and the role of finite-measure or uniformly
tight supports.

## Files

- `main.tex` - paper source
- `meeting_summary.tex` - short meeting-ready summary of the results and proof ideas
- `README.md` - project notes
- `.gitignore` - ignores LaTeX build output
- `Makefile` - optional build commands

## Build

If you have a LaTeX distribution installed, run:

```sh
make
```

or directly:

```sh
pdflatex main.tex
pdflatex main.tex
```
